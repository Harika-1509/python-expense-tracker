# Python Expense Tracker

A simple and beginner-friendly Expense Tracker application built using Python.  
This project helps users manage daily expenses, track spending records, and practice core Python programming concepts.

---

## Features

- Add new expenses
- View all expenses
- Store expense records using JSON/file handling
- Beginner-friendly command line interface
- Simple and clean project structure

---

## Technologies Used

- Python
- JSON
- File Handling

---

## Project Structure

```bash
expense_tracker/
│── main.py
│── expenses.json
│── README.md
```

---

## How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/Harika-1509/python-expense-tracker.git
```

### 2. Open Project Folder

```bash
cd python-expense-tracker
```

### 3. Run the Program

```bash
python main.py
```
